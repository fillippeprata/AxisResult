# AxisResult

> **This repository has moved.** AxisResult is now part of the **Axis Framework**.

AxisResult began here as a standalone, zero-dependency *Result monad* for C#
(Railway-Oriented Programming). It has since become one of the foundations of the
**Axis Framework** — which was kept **private during its validation period** and is
now public at:

### 👉 [axis-sys/axis-framework-dotnet](https://github.com/axis-sys/axis-framework-dotnet)

- **Source:** [`src/0-Foundations/AxisResult/`](https://github.com/axis-sys/axis-framework-dotnet/tree/main/src/0-Foundations/AxisResult)
- **Docs:** [English](https://github.com/axis-sys/axis-framework-dotnet/tree/main/docs/en-us/0-Foundations/AxisResult) · [Português](https://github.com/axis-sys/axis-framework-dotnet/tree/main/docs/pt-br/0-Foundations/AxisResult)
- **NuGet:** [`AxisResult`](https://www.nuget.org/packages/AxisResult) — same package; versions `1.3.0` and later are published from the framework repository.

The **tags and releases** in this repository (up to `v1.2.0`) are kept as the historical
source of the AxisResult NuGet versions `1.0.0`–`1.2.0`.
